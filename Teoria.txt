git y github son herramientas generalmente usadas por desarrolladores de software y programadores creativos
git es un software de control de versiones que tambien puede incluir colaboraciones
github es un sitio web en el cual se realizan proyectos que son administrados por git o que possen un control de versiones
un repositorio es donde se guardan los proyectos, hay commit new file y commit changes
hash es un identificador de un commit algo asi como un curp
branching: son ramas para un posible cambio en el archivo, son ideas que van surguiendo y no quieres que se olviden
pull request: es una peticion que el propietario de un fork de un repositiorio hace al propietario del repositorio original para que este ultimo agregue los commits que hay en el fork
marge: es la confirmacion o fusion de las modificaciones hehcas en el fork
lenguajes mas usados son: java script, python, java, ruby, php, c++

un fork es la copia identica de un repositorio original
